LET'S GET THIS BREAD


a chemistry project by Isabella Hochschild

The objective of this project was to analyze the chemical functions of ingredients in bread and determine which leavening agent (no agent, yeast, yeast w/ honey, sourdough starter, baking soda, baking powder) produces the highest rise of bread dough.

This project was created using Flask, a microframework for Python, and HTML templates supplemented by JavaScript and CSS. An additional coding language used was A-Frame.
